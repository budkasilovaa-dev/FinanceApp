# FinanceApp

Приложение для управления личными финансами.

## Возможности
- Добавление и хранение кошельков (Wallets)
- Добавление транзакций (Transaction) для кошельков
- Подсчет баланса по кошелькам
- Сохранение данных в `data.json`
- Юнит-тесты для проверки логики приложения (xUnit)

## Структура проекта
- `FinanceApp/` – основной проект
- `FinanceApp.Tests/` – юнит-тесты проекта

## Технологии
- .NET 9
- C#
- xUnit для тестирования

## Как запустить
```bash
dotnet run --project FinanceApp
